# ska-dsh_lmc
SKA Dish LMC Control System Prototype

**Getting Started**  
___
Install the project dependencies:  
	- omniORB [omniorb.sourceforge.net/]  
	- zmq [http://zeromq.org/]  
	- Tango Control System [http://www.tango-controls.org/]  
	- jsoncpp  
	- pugixml  
	- cmake (>=2.8) [https://cmake.org/]
